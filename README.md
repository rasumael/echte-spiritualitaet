# Echte Spiritualität
## Was ist wahre oder echte Spiritualität?

Echte Spiritualität kann nur aus dem Inneren kommen.
Sie hat die Aufgabe uns zu zeigen wie großartig wir und unsere eigene Seele sind.
Sie führt uns in die vorbestimmte Richtung.
Ein freier Wille ist nämlich eine Illusion.
Unsere Körper sind komplett von außen manipulierbar - was wir fühlen, denken, erkennen, erinnern, und wie wir uns bewegen. 
Im Idealfall übernimmt das nur die Urquelle - unsere Sonne. Auch unsere Seele ist von ihr gesteuert.
